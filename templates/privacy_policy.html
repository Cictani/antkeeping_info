Privacy policy of website antkeeping.info

Webserver logs

What personal data is collected?

When you access this website, the following information is stored in a server log file:

  • The page you visited
  • Date/Time of visit
  • The amount of data sent in Byte
  • Source/Link from which you accessed the page
  • The web browser you used
  • The operating system you used
  • Your IP-address

What is this data used for?

The data is used for two things:

  1. Protection and prevention of attacks against this website like DDoS, Brute force password attacks etc. Since it might be neccessary to conduct legal actions it is necessary to store the visitor's IP address.
  2. Creating anonymized web site statistics

How long is your personal data stored?

The personal data in the server log files is stored for a maximum of two weeks. After that your IP-address is anonymized.

The website statics are stored for an unlimited time. But since all the data used for it was anonymized previously it is not possible to make any connection to your person.
